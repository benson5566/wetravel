# WeTravel 雲端直連版 ✈️

這是一個專為雙人/小團體設計的旅遊規劃 APP，特色是支援**即時協作**、**離線瀏覽**與**記帳分帳**。

## ✨ 主要功能

- 📅 **行程規劃**：拖拉式介面，輕鬆安排每日景點與交通。
- ☁️ **雲端同步**：透過 Google Firebase 實現多人即時協作 (一人修改，其他人秒同步)。
- 💰 **記帳分帳**：自動計算總花費與個人應付金額，支援多國貨幣匯率換算。
- 📍 **口袋名單**：蒐集想去的地點，支援 Google Maps / Naver Map (韓國) / 高德地圖 (中國) 導航。
- 📱 **PWA 支援**：可安裝至手機主畫面，並支援離線查看行程。

## 🚀 如何使用 (部署教學)

本專案是一個純靜態網頁 (Static Web App)，您可以使用 **GitHub Pages** 或 **Firebase Hosting** 免費託管。

### 方法一：使用 GitHub Pages (推薦)
1.  將此資料夾的所有檔案上傳到 GitHub Repository。
2.  進入 Repo 的 `Settings` > `Pages`。
3.  在 `Source` 選擇 `Deploy from a branch` (通常是 `main` 或 `master` branch)。
4.  儲存後，等待約 1-2 分鐘，您就會獲得一個專屬的網址 (例如 `https://yourname.github.io/wetravel/`)。
5.  將該網址分享給旅伴，即可開始協作！

### 方法二：本機測試
1.  直接用瀏覽器開啟 `index.html`。
    - *注意*：本機開啟時，PWA 安裝功能可能會受限，建議使用 Web Server (如 VS Code 的 Live Server 套件) 開啟。

## 🛠️ 開發者資訊
- **前端核心**：Vue 3 (Composition API)
- **樣式框架**：Tailwind CSS (CDN)
- **後端服務**：Google Firebase (Firestore + Auth)
- **圖標庫**：Phosphor Icons

## 📱 安裝到手機 (PWA)
1.  使用手機瀏覽器 (Safari / Chrome) 開啟部署後的網址。
2.  點選「分享」按鈕 (iOS) 或選單 (Android)。
3.  選擇「加入主畫面 (Add to Home Screen)」。
4.  現在它就像一個原生 APP 一樣可以離線使用了！
